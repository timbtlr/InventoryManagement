The Rundown

This application helps track and manage inventory by allowing users to add and subtract quantities for different parts. Each part has the following attributes:

  • Part number - The unique part identifer in the inventory
  • Description - A simple description of the part
  • Cost - The dollar cost per part
  • Quantity - The number of parts currently held within inventory
  • UOM - Unit of Measurement used to measure the part
  • Children - Any parts that are used to create this part
Parts can be added to the inventory at will and quantities can be removed or added as they are moved in and out of physical space.

The Benefits

  • Add/Remove inventory items
  • See current inventory value
  • Create parts with a bill of materials
  • Automatically adjust inventory based on material requirements
  • Help Predict purchasing requirements
